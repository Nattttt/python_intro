PREGUNTA 1

Devuelva la suma de todos los números de una lista, ignorando las secciones que empiecen con 6 y terminen con 7.

>>> suma([1, 2, 4])
7
>>> suma([0, 6, 4])
10
>>> suma([1, 6, 4, 0, 7, 2])
3
PREGUNTA 2

Dada una lista, devuelva si la misma contiene un 2 seguido de otro 2

>>> tiene_2_2([1, 2, 2, 4, 5])
True
>>> tiene_2_2([1, 2, 0, 4, 2])
False
>>> tiene_2_2([2, 1, 0, 5, 1])
False
PREGUNTA 3

Verifique si en una lista dada aparece la misma cantidad de veces las cadenas "pi" y "py"

>>> contador_pi_py("xpymlkpewqpi")
True
>>> contador_pi_py("xpypymlkpewqpildpi")
True
>>> contador_pi_py("xpymlkpewqpildpi")
False
PREGUNTA 4

Dadas dos cadenas, verifique si cualquiera de las dos aparece al final de la otra.

>>> es_cola("Hiabc", "abc")
True
>>> es_cola("AbC", "HiaBc")
True
>>> es_cola("abc", "abXabc")
